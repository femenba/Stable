## CTA Strategy

### CTA Hierarchy (strongest to softest)
1. **Save this** — best for carousels and education content (saves = algorithmic gold)
2. **Send this to someone** — best for validation posts (reach multiplier)
3. **Drop it below** — best for question-based posts (comments boost reach)
4. **Try it free** — product CTAs, always with link framing
5. **Link in bio** — keep this for posts where the app is explicitly the answer

### CTA Rules
- One CTA per post. Never two.
- Never "like and follow" — it reads as desperate and is penalised
- Never "smash the notification bell" — wrong brand voice
- Always phrase CTAs as invitations, not commands

### CTA Phrasing Bank
```
→ save this for the next time it happens
→ send this to the person who needs to hear it
→ what's your experience? drop it below
→ try it free — link in bio
→ which one hits hardest? vote below
→ share this with someone who'd say "this is so me"
→ reply with your version of this
→ does this land for you?
→ what would you add?
→ follow for more of this
```

## Reusable AI Prompts for Content Generation

### Prompt 01 — Generate Validation Caption
```
You are the creative director of @stableadhd, a premium ADHD productivity app.

Write an Instagram caption for a validation post. The post image shows a quote about [SPECIFIC ADHD EXPERIENCE].

Voice requirements:
- Calm, precise, warm, never preachy
- No toxic positivity
- No hustle culture language
- Like a trusted friend who understands neuroscience
- Short paragraphs (1-3 lines each)
- End with one specific, soft CTA

Brand context: stable. is for adults with ADHD who are done with shame and chaos. The aesthetic is minimalist, dark green/cream, Apple-quality design.

The specific topic is: [TOPIC]
The CTA should be: [SAVE / SHARE / COMMENT / LINK]

Produce 3 versions. Each under 150 words.
```

### Prompt 02 — Generate Educational Carousel Slides
```
You are the creative director of @stableadhd, a premium ADHD productivity app.

Create the text content for an Instagram carousel. 7 slides total.

Topic: [TOPIC]
Angle: [EDUCATION / MYTH-BUSTING / HOW-TO / SCIENCE]

Requirements:
- Slide 1: Bold, provocative cover title (under 8 words)
- Slides 2-6: One clear insight per slide. Short paragraphs. 40-60 words max per slide.
- Slide 7: Emotional close + CTA to try stable. free

Tone: Like reading a premium wellness publication.

Do not use: "Let's dive in!" / "Here's what you need to know:" / "In conclusion:"

Topic: [TOPIC]
```

### Prompt 03 — Generate Reel Script
```
You are the creative director of @stableadhd.

Write a 20-30 second Instagram Reel script. Text-on-screen format OR voiceover format (specify which).

Hook requirement: First 2 seconds must stop the scroll. Use a specific, relatable ADHD moment — not a question.

Structure:
- Hook (2s): Visual or statement that lands immediately
- Problem (6s): Name the ADHD experience precisely
- Insight (8s): The reframe or education piece
- Resolution (8s): How stable. helps (never hard-sell)
- CTA (3s): One action

Voice: Calm, measured, intelligent. No filler words. No "hey guys."
Audio direction: [ambient / lo-fi / silence / trending audio category]

Topic: [TOPIC]
```
