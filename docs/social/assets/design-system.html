Grain Texture Rule

Every post on a dark background gets a grain overlay at 3–5% opacity. In Canva: add a "Static" noise texture element → set blend mode to Overlay → set opacity to 4%.

This single treatment is what separates premium visual work from flat digital graphics. It adds depth and a tactile, film-quality feel.

Light posts (Warm White bg): use 6% grain · Rose posts: 4% grain
