## Layout Rules

- **Minimum 15% safe zone margin** on all sides — Instagram crops edges on some placements
- **Content should never feel crowded** — ADHD brains need visual breathing room
- **8pt grid system** for all layout decisions (8, 16, 24, 32, 48, 64, 80)
- **Whitespace is design** — if it feels empty, it's working

## Canva → Meta Business Suite Pipeline

1. Design in Canva using Brand Kit
2. Export PNG (feed) or MP4 (reels) at original size
3. Open Meta Business Suite → Planner → Create Post
4. Select account: @stableadhd
5. Upload file
6. Paste caption from captions-bank.md
7. Add hashtags from hashtag-bank.md (after line break in caption)
8. Set date and time from posting-calendar.md
9. Publish or Schedule

**Time cost per post:** ~10 minutes with ready templates.
**Time cost per week (7 posts + stories):** ~45 minutes if templates are set up.

## Visual Identity Rules (summary)

### What to use
- Motifs: soft morning light, plants, hands holding a phone, close-ups of objects
- Textures: linen, paper, aged wood, stone
- If using people: candid, not posed — never smiling at camera

### What never to use
- Bright or saturated colors (no neon, no primary RGB)
- Gradients that look like tech startups from 2019
- White backgrounds with black text (too clinical)
- Any shade that reads as "childish" or "playful"
- Stock photography
- Staged "productive person at desk" imagery
- Mascots, characters, or cartoon ADHD brains
- Clipart or icon-heavy designs

### Grain overlay
Every post gets a subtle grain overlay at 3–5% noise, multiply blend mode. This adds a premium film quality and distinguishes stable. from flat digital designs.
